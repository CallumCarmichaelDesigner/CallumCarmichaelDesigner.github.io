McQueen Fashion
User Interface & Experience Design

Overview

McQueen Fashion provides the latest in men’s styles and fashion trends.
The iOS application acts as a luxury fashion resource for males.

Case Study

McQueen Fashion is an application designed to serve as a luxury fashion resources for males. The application provides the user with a multitude of features and is tailored towards each individual users taste.

Discover a wide selection of luxury fashion products featuring brands like Alexander McQueen, Gucci and Levis. Discover new outfits everywhere with the visual search camera and connect with users in a fashion focused community. Shop and discover the latest men’s fashion trends and enjoy personalised fashion inspiration with a curated homepage feed.

User Interface

The interface was designed to provide the user with a simple yet luxurious experience. The app was designed with a clear and concise style, a subtle colour palette and clean lines combined with slick typography constructs an elegant simplicity that is aligned with a luxurious brand identity. Furthermore, this minimalistic style draws attention to product imagery and information allowing users to focus on achieving their primary goal – discovering fashionable products.

The screen is void of extraneous and confusing details which allows the user to comprehend the purpose of each element. The clarity and consistency of the interface inspires confidence within the user and leads to further use and engagement. Overall, the core interface is simple and functional. Every screen supports a single strong action and has been designed to optimise user satisfaction.

Animated Design

The application provides essential content and functionalities combined with coherent, consistent and intuitive navigation. The user experience was built around adaption towards user needs. The animation within the application and process of interaction was developed to act as a functional element.

The functional animation helps achieve the goal of creating an intuitive, responsive and human interface. Simple and natural animations were created to provide information and ease of interaction along with a stylish effect that provides a positive user experience. Animated transitions help the user understand change by following motion and visual feedback shows that the application has received the user’s actions. All the animations have a clear purpose, the animated and interactive interface enhances user experience by helping guide user flow.
